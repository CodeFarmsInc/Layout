This directory provides results of those examples
marked as OK in the table shown in test\readme.txt,
or which demonstrate some of the problems of the existing version.

To see the svg result, click in Windows
on any of the files out*.svg.
To see the Java results, run any of the files jav*.bat.

For example, out10.svg is the svg result of inp10,
             jav21.bat if for the Java display of inp21.
